# proyecto-de-graduacion
va a tratar de resolver el cambio climtico
## Esta es la idea de mi proyecto:
> crearia una pagina web sobre formas de cuidar el planeta y un fondo de donaciones  para ayudar a cambiar el cabio climatico (paneles solareas, botes de basura recyclable y lugares que convivan con el medio ambiente
> me gustaria presentar este trabajo con pagina web y talvez vision por ordenador


>EcoFuture es una plataforma web creada con el objetivo de promover el cuidado del medio ambiente y concienciar a las personas sobre la importancia de proteger nuestro planeta.

El proyecto fue desarrollado utilizando HTML para la estructura de la página, CSS para el diseño y la apariencia visual, JavaScript para agregar interactividad y Flask, un framework de Python, para gestionar el funcionamiento del sitio web.

La página cuenta con un diseño moderno y fácil de utilizar. En ella los visitantes pueden encontrar información sobre problemas ambientales como la contaminación, el cambio climático y la importancia del reciclaje. Además, se presentan diferentes acciones que cualquier persona puede realizar para ayudar al planeta, como reducir el uso de plásticos, ahorrar energía y proteger los recursos naturales.

Una de las funciones principales de EcoFuture es su sección de donaciones, donde se explica cómo los aportes pueden destinarse a proyectos como la instalación de paneles solares, la creación de áreas de reciclaje y otras iniciativas enfocadas en mejorar el medio ambiente.

Durante el desarrollo del proyecto se buscó crear una experiencia agradable para el usuario mediante un diseño atractivo, imágenes relacionadas con la naturaleza, una navegación sencilla y una estructura organizada para que la información fuera clara y fácil de entender.

En conclusión, EcoFuture no solo representa un proyecto de programación, sino también una propuesta para inspirar a las personas a tomar acciones que contribuyan a un futuro más sostenible. El proyecto demuestra cómo la tecnología puede utilizarse para educar, informar y motivar cambios positivos en la sociedad.
